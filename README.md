### Running the Backend

Setup Virtual environment using <br>
`python3 -m venv venv`<br>

Activate venv <br>
`source venv/bin/activate`<br>

Install Dependencies <br>
`pip3 install -r requirements.txt`

Runserver using<br>
`python3 manage.py runserver`<br>
